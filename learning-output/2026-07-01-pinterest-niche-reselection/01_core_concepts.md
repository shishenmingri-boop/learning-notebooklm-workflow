# 重要用語と評価軸の定義

## Pinterest/note/アフィリの仕組み用語集

### Pinterest（ビジュアル検索エンジン）
タイムライン型SNSではなく検索エンジンとして機能する。1つのピンが半年〜1年単位で流入を生み続ける「常緑（evergreen）」性質を持つ。表示単価は払われず、収益はすべて外部送客（ブログ/note）経由のアフィリエイトに依存する。

### AI修正画像ラベル（Gen AIラベル）
2025年春から導入されたPinterestの機能。AI生成・編集されたと判定された画像ピンに自動でラベルを表示する。IPTC写真メタデータ規格とPinterest独自のAI分類器を併用して判定する。

### AI生成コンテンツの表示調整機能
2025年10月導入。**美容・アート・ファッション・インテリア**など、生成AIの使用頻度が高いカテゴリにおいて、ユーザーが「このカテゴリのAI生成コンテンツを少なく表示」に設定できる。設定は任意だが、該当カテゴリのAI生成ピンは一部ユーザーへのリーチが構造的に低下するリスクがある。

### 直リンク可否
楽天アフィリエイトはPinterestへの直接リンクが許可されているが、Amazonアソシエイトや多くのASP（A8.net等）は規約上、ブログ/note等を経由する「ワンクッション」を要求する場合が多い。ニッチによって使えるASPの組み合わせが変わる。

### 紹介料率（アフィリ単価の源泉）
楽天・Amazonともに商品ジャンルごとに固定の基本紹介料率が決まっている。個別商品で料率アップ（楽天は最大20%）が設定される場合もあるが、ジャンル基本料率がニッチ選定の土台になる。

### note有料記事の実用ノウハウ系/読み物系
noteの公式分析（2026年1月）による分類。実用ノウハウ系（投資・スキル・課題解決）は平均単価1,842円で、読み物系（983円）の約1.9倍。文字数と売上の相関はほぼゼロ（-0.023）であり、量より「誰にしか語れないか」が重要。

### 薬機法・景品表示法
医薬品ではない化粧品・健康食品等について、医薬品的な効能効果を暗示する表現を禁止する法律。アフィリエイターにも責任が及ぶ可能性があり、2016年の消費者庁通達以降、アフィリエイトサイトも規制対象と明確化されている。

## 評価軸の定義（ニッチ比較で使う8軸）

| 軸 | 定義 | 高スコアの意味 |
|---|---|---|
| A. インプレッション獲得しやすさ | 画像の保存率・視覚的訴求力 | 保存されやすい・目を引く画像が作りやすい |
| B. 検索ボリューム/ロングテール性 | Pinterest内検索・note内検索でのニーズの厚み | ロングテールキーワードが豊富にある |
| C. アフィリ料率 | 楽天/Amazonの基本紹介料率 | 料率が高い（同じ成約でも報酬が高い） |
| D. note記事との相性 | 比較記事・体験談・ノウハウ記事化のしやすさ | note公式の急上昇カテゴリとの重なりが大きい |
| E. 競合密度の低さ | レッドオーシャン度の逆数 | 発信者が少なく差別化しやすい |
| F. Flux画像生成との相性 | no-people/no-text条件で成立する構図か | 無人・非顔写真でも訴求力を保てる |
| G. 法規制リスクの低さ | 薬機法・景品表示法等の表現規制の強さの逆数 | 表現規制が緩く運用しやすい |
| H. 既存資産の転用可能性 | 収納パイプラインのコード・プロンプト・記事フォーマットの再利用度 | 移行コストが低い |

各軸は1〜5点。8軸合計40点満点でスコアリングする（`02_deep_dive.md`参照）。

## よくある誤解

- **「ソロキャンプが伸びている」という誤解**: 2026年のデータでは、ソロ向けキャンプ用品の取引は前年比-12.4%減少しており、伸びているのは2〜3人向け（+9.4%）・4人以上向け（+16.6%）の**ファミリーキャンプ**である。ソロキャンプ単体をニッチにするのは古い前提に基づくリスクがある。
- **「アフィリ料率は些細な差」という誤解**: 収納(3%)とペット/美容(5%)では、同じ成約1件でも報酬が1.67倍異なる。月間成約数が同じでも収益は大きく変わるため、料率差は軽視できない。
- **「AI生成画像なら何のジャンルでも同じように使える」という誤解**: Pinterestは2025年10月から美容・アート・ファッション・インテリアで「AI生成コンテンツを少なく表示」オプションを提供しており、これらのカテゴリはAI生成主体のパイプラインにとって構造的なリーチ低下リスクを負う。
- **「薄利多売のニッチは避けるべき」という誤解の裏返し**: 収納ニッチは料率が低いが、既存資産（30ピン・10note・自動化コード一式）の転用可能性が非常に高く、ゼロから始めるより実行速度が速いという別の強みがある。単純な料率比較だけでニッチを決めるのは早計。
