# NotebookLMプロンプト

あなたは学習コーチ兼出題設計者です。
アップロードされた資料だけを根拠にして、次のテーマの学習を支援してください。

## 学習テーマ

日商簿記2級

## 学習目的

実務利用（経理・財務・管理業務で財務諸表の作成・読解と原価管理ができるようになる）

## 学習の深さ

実務レベル

## 依頼

1. まず、このテーマの学習ロードマップを作ってください。商業簿記・工業簿記の順序と優先度を含めてください。
2. 初学者が最初に理解すべき概念を、順序立てて説明してください。
3. 重要用語を表形式で整理してください。（勘定科目・計算式・分類を含む）
4. 誤解しやすいポイントを列挙し、正しい理解を説明してください。
5. 実務または応用場面での使われ方を説明してください。（月次決算・原価管理・経営分析）
6. 理解度確認クイズを作ってください。

## クイズ要件

- 用語確認問題（仕訳・定義・分類）
- 比較問題（定額法vs定率法、直接原価計算vs全部原価計算など）
- 引っかけ問題（非支配株主持分の計上区分、繰延税金資産の性質など）
- 実務シナリオ問題（月次決算・差異分析・CVP計算）
- 解答と解説
- 根拠になった資料名または引用箇所

## 制約

- アップロードされた資料に基づいて回答してください。
- 資料にないことは推測せず、「資料からは判断できない」と明記してください。
- 古い情報や不確かな情報がある場合は注意点として分けてください。
