# 日商簿記2級 深掘り解説

## 1. 連結会計の仕組みと実務

### なぜ連結財務諸表が必要か

親会社が子会社に売上を立てれば、単体財務諸表上では利益が生まれる。しかしグループ全体では何も実現していない。連結財務諸表はこの「グループ内取引」を消去し、外部との取引だけを反映させる。

### タイムテーブル（連結会計の解法）

支配獲得時から時系列で純資産の変動を追う表を作ると、開始仕訳が機械的に導ける。

```
                   支配獲得時     ×1年度末
資本金              ×,×××      ×,×××（変化なし）
利益剰余金           ×,×××      ×,×××（純利益分増加）
非支配株主持分       ×,×××      ×,×××（利益配分後）
のれん              ×,×××      ×,×××（償却後）
```

### よくある失敗
- **開始仕訳を忘れる**：2年目以降は前期末の連結修正仕訳を「開始仕訳」として再掲する
- **のれんの償却期間を間違える**：企業結合会計基準では20年以内の定額法が原則
- **内部利益の消去を当期分だけやる**：前期末在庫の内部利益消去（「開始仕訳」）も必要

## 2. 標準原価計算と差異分析の実務活用

### 製造間接費のシュラッター図

```
実際操業度          標準操業度
    |                   |
    ↓                   ↓
固定費線 ーーーーーーーーーーーーー 基準操業度点
    |        ↑              ↑
    |   操業度差異       能率差異
    ↓
実際発生額点
    ↑
予算差異
```

**三分法（能率差異を変動費のみで計算）：**
- 予算差異 = 予算許容額 − 実際発生額
- 操業度差異 = 固定費率 × （実際操業度 − 基準操業度）
- 能率差異 = 変動費率 × （標準操業度 − 実際操業度）

**四分法（能率差異を変動費・固定費に分解）：**
- 固定能率差異が追加される

### 実務での使われ方
標準原価計算は**原価管理**が目的。差異を発見し、次月の改善につなげる。
- 価格差異が大きい → 購買部門への価格交渉指示
- 能率差異が大きい → 生産現場の作業改善
- 操業度差異が大きい → 設備稼働率の見直し

## 3. 税効果会計の仕組み

### 会計利益と税務上の課税所得のズレ

```
会計利益（税引前）：1,000万円
　+ 損金不算入（貸倒引当金超過）：100万円
　= 課税所得：1,100万円

法人税等（実際）：1,100 × 30% = 330万円
本来あるべき税金：1,000 × 30% = 300万円
繰延税金資産：100 × 30% = 30万円
```

### 実務での注意点
- **回収可能性の検討**：将来の課税所得が十分に見込めない場合、繰延税金資産は計上できない（評価性引当額の設定）
- **法定実効税率の変更**：税率が変わると既計上の繰延税金資産・負債の金額を修正する

## 4. CVP分析と経営意思決定

### 損益分岐点の図解

```
      売上高
    ↗（傾きは1）
   /
  /   ← 利益ゾーン（この面積が安全余裕）
 /
損益分岐点
 \
  \   ← 損失ゾーン
   \
    総費用（固定費 + 変動費）
```

### 複数製品のCVP分析（セールスミックス）
製品ごとに貢献利益率が異なる場合、「加重平均貢献利益率」を使って計算する。

### 直接原価計算 vs 全部原価計算
| 項目 | 全部原価計算 | 直接原価計算 |
|------|------------|------------|
| 固定製造間接費の扱い | 製品原価（期末在庫に含む） | 期間費用（売上原価） |
| 利益の違い | 在庫増加時に利益大 | 在庫増加しても利益不変 |
| 用途 | 外部報告（財務諸表） | 内部管理（意思決定） |

**直接原価計算から全部原価計算への利益調整：**
```
直接原価計算の利益
　± 期末在庫の固定費 − 期首在庫の固定費
= 全部原価計算の利益
```

## 5. リース会計

### ファイナンス・リースの判定基準

次の**いずれか**を満たす場合、ファイナンス・リース（資産計上必須）：
1. **ノンキャンセラブル**：解約不能
2. **フルペイアウト**：リース料総額がリース資産の経済的利益のほぼ全部（90%以上）または耐用年数の75%以上

**仕訳（利息法）：**
```
リース開始時：（借）リース資産 ×× ／（貸）リース債務 ××
支払時：（借）リース債務 ×× 支払利息 ×× ／（貸）現金 ××
決算：（借）減価償却費 ×× ／（貸）リース資産減価償却累計額 ××
```

## 6. 外貨換算会計

### 換算レートの使い分け

| 項目 | 換算レート |
|------|----------|
| 外貨建債権・債務（流動） | 決算時のレート（CR） |
| 外貨建有価証券（売買目的） | 決算時のレート（CR） |
| 固定資産・払込資本 | 取得時のレート（HR） |
| 売上・仕入（取引時） | 取引時のレート（HR） |

**為替差損益の計上：**
```
期末評価で円安の場合（外貨建売掛金）：（借）売掛金 ×× ／（貸）為替差益 ××
期末評価で円高の場合：（借）為替差損 ×× ／（貸）売掛金 ××
```

## 設計上の注意点・よくある失敗

| 論点 | よくある失敗 | 対策 |
|------|------------|------|
| 連結 | 開始仕訳の作成漏れ | タイムテーブルで毎回確認 |
| 税効果 | 法人税等調整額の貸借を逆にする | 「繰延税金資産は資産＝前払い」で覚える |
| 標準原価 | 差異の有利・不利を逆にする | 標準−実際がプラス＝有利で統一 |
| 総合原価計算 | 加工進捗度を材料費にも適用する | 材料費は始点投入なら進捗度1.0 |
| CVP | 変動費率と貢献利益率を足して100%確認 | 変動費率 + 貢献利益率 = 1 |
