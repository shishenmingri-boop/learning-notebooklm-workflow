# 日商簿記2級 重要用語・コア概念

## 商業簿記

### 有価証券の4分類

| 分類 | 評価方法 | 評価差額の処理 |
|------|----------|----------------|
| 売買目的有価証券 | 時価法 | 有価証券評価損益（当期損益） |
| 満期保有目的債券 | 償却原価法 | 有価証券利息 |
| 子会社株式・関連会社株式 | 原価法 | なし |
| その他有価証券 | 時価法 | その他有価証券評価差額金（純資産、税効果適用） |

**よくある誤解：** 「その他有価証券」の評価差額は損益計算書ではなく貸借対照表の純資産に計上する（全部純資産直入法）。

---

### 減価償却の3方法

| 方法 | 計算式 | 特徴 |
|------|--------|------|
| 定額法 | （取得原価 − 残存価額）÷ 耐用年数 | 毎期均等額 |
| 定率法 | 期首帳簿価額 × 償却率 | 初期に多く費用計上 |
| 生産高比例法 | （取得原価 − 残存価額）× 当期利用量 ÷ 総利用可能量 | 航空機・鉱山等 |

**注意：** 残存価額は現行の会計基準では0円（備忘価額1円）が多い。

---

### 引当金の要件（4要件）

1. 将来の特定の費用・損失であること
2. 発生が当期以前の事象に起因すること
3. 発生の可能性が高いこと
4. 金額が合理的に見積もれること

**主な引当金と計上仕訳：**

```
貸倒引当金：（借）貸倒引当金繰入 ／（貸）貸倒引当金
退職給付引当金：（借）退職給付費用 ／（貸）退職給付引当金
修繕引当金：（借）修繕引当金繰入 ／（貸）修繕引当金
```

---

### 税効果会計

**一時差異の種類：**

| 差異 | 内容 | 処理 |
|------|------|------|
| 将来減算一時差異 | 会計＞税務（費用）。将来税金が減る | 繰延税金資産（資産） |
| 将来加算一時差異 | 会計＜税務（収益）。将来税金が増える | 繰延税金負債（負債） |

**仕訳パターン（繰延税金資産の計上・解消）：**
```
計上時：（借）繰延税金資産 ／（貸）法人税等調整額
解消時：（借）法人税等調整額 ／（貸）繰延税金資産
```

**簿記2級で出る論点：**
- 減価償却費の償却限度超過額
- 貸倒引当金の繰入限度超過額
- その他有価証券の評価差額（→繰延税金負債）

### 連結会計の基本仕訳

**支配獲得時（資本連結）：**
```
（借）資本金         ×××
     資本剰余金      ×××
     利益剰余金      ×××
     のれん          ×××  ← 投資額 - 子会社純資産持分の差額
（貸）子会社株式     ×××
     非支配株主持分  ×××  ← 子会社純資産 × 非支配株主比率
```

**翌期以降（のれん償却）：**
```
（借）のれん償却 ×× ／（貸）のれん ××
```

**子会社利益の非支配株主への配分：**
```
（借）非支配株主に帰属する当期純利益 ×× ／（貸）非支配株主持分 ××
```

**内部取引の消去（成果連結）：**
```
（借）売上高 ×× ／（貸）売上原価 ××
（借）売上原価 ×× ／（貸）商品 ××  ← 期末在庫の未実現利益
```

## 工業簿記

### 原価計算の流れ（勘定連絡図）

```
材料      →  仕掛品  →  製品  →  売上原価
労務費    ↗
製造間接費 ↗（配賦）
```

**費目別計算：**
- 材料費 = 月初棚卸 + 当月仕入 − 月末棚卸
- 製造間接費は「予定配賦率 × 実際操業度」で配賦

---

### 総合原価計算の計算方法

| 方法 | 月末仕掛品の計算 | 特徴 |
|------|----------------|------|
| 平均法 | （月初仕掛品原価 + 当月製造費用）÷ 完成品換算量合計 | 平均単価で計算 |
| 先入先出法 | 当月製造費用 ÷ 当月投入完成品換算量 | 先に入ったものが先に完成 |

**完成品換算量（加工費）= 月末数量 × 加工進捗度**

---

### 標準原価計算の差異分析

**直接材料費差異：**
```
価格差異 = （標準単価 − 実際単価）× 実際消費量
数量差異 = 標準単価 × （標準消費量 − 実際消費量）
```

**直接労務費差異：**
```
賃率差異 = （標準賃率 − 実際賃率）× 実際就業時間
時間差異 = 標準賃率 × （標準就業時間 − 実際就業時間）
```

**製造間接費差異（四分法）：**
```
予算差異   = 予算許容額 − 実際発生額
操業度差異 = 固定費率 × （実際操業度 − 基準操業度）
能率差異   = 変動費率 × （標準操業度 − 実際操業度）
（固定能率差異 = 固定費率 × （標準操業度 − 実際操業度））
```

**有利差異 vs 不利差異：** 標準 − 実際 がプラスなら有利差異（コスト削減）、マイナスなら不利差異（コスト超過）。

### CVP分析の計算式

| 指標 | 計算式 |
|------|--------|
| 貢献利益（限界利益） | 売上高 − 変動費 |
| 貢献利益率 | 貢献利益 ÷ 売上高 |
| 損益分岐点売上高 | 固定費 ÷ 貢献利益率 |
| 損益分岐点販売量 | 固定費 ÷ 単位あたり貢献利益 |
| 安全余裕率 | （実際売上高 − 損益分岐点売上高）÷ 実際売上高 |
| 目標利益達成売上高 | （固定費 + 目標利益）÷ 貢献利益率 |

## 誤解しやすいポイント TOP5

1. **のれんは「無形固定資産」**：投資有価証券ではない
2. **非支配株主持分は「純資産の部」**：負債ではない
3. **繰延税金資産は「資産」**：費用ではない（「繰延」は費用でも資産でもよく混乱する）
4. **その他有価証券の評価差額は税効果適用後に純資産へ**：税効果なしで純資産計上ではない
5. **直接原価計算は外部報告に使えない**：管理目的専用（固定費を製品原価に含めないため）
