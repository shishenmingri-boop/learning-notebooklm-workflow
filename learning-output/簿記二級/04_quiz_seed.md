## 仕訳問題（正誤・選択式）

### 問題1：有価証券の評価
当社が保有する「その他有価証券」（取得原価100万円）の期末時価が120万円になった。
法定実効税率30%として、期末の仕訳を示せ。

**正解仕訳：**
```
（借）投資有価証券 200,000 ／（貸）その他有価証券評価差額金 140,000
                              繰延税金負債 60,000
```

**引っかけポイント：** 評価差額を損益計算書に計上してはいけない（純資産直入法）。税効果も適用する。

---

### 問題2：連結修正仕訳（のれん償却）
P社はS社の株式80%を1,000万円で取得した（支配獲得時のS社純資産：1,100万円）。
のれんを10年定額法で償却する場合の第1期末の仕訳を示せ。

**のれん計算：** 1,000 − 1,100 × 80% = 1,000 − 880 = 120万円
**非支配株主持分：** 1,100 × 20% = 220万円
**当期償却額：** 120 ÷ 10 = 12万円

**正解仕訳：**
```
（借）のれん償却 120,000 ／（貸）のれん 120,000
```

### 問題3：標準原価計算（差異分析）
- 標準単価：500円、標準消費量：10kg
- 実際単価：520円、実際消費量：9.5kg

直接材料費差異、価格差異、数量差異を求めよ。

**正解：**
```
直接材料費差異 = 標準（500×10=5,000）− 実際（520×9.5=4,940）= +60円（有利）
数量差異 = 500 ×（10 − 9.5）= +250円（有利）
価格差異 = （500 − 520）× 9.5 = −190円（不利）
```

**引っかけポイント：** 「価格差異の計算に実際消費量を使う」ことを忘れやすい。

### 問題4：CVP分析
- 販売単価：2,000円
- 変動費率：60%
- 固定費：400,000円

損益分岐点売上高と、目標利益200,000円を達成するための売上高を求めよ。

**正解：**
```
貢献利益率 = 1 − 0.6 = 40%
損益分岐点売上高 = 400,000 ÷ 0.4 = 1,000,000円
目標利益達成売上高 = （400,000 + 200,000）÷ 0.4 = 1,500,000円
```

## 引っかけ問題

1. 「繰延税金資産は費用性資産だから損益計算書に計上する」→ **誤り**（貸借対照表の資産）
2. 「非支配株主持分は少数株主の持分だから負債に計上する」→ **誤り**（純資産の部）
3. 「のれんは20年で必ず償却しなければならない」→ **誤り**（20年以内であれば何年でもよい）
4. 「直接原価計算は外部報告（財務諸表）に使える」→ **誤り**（管理目的のみ）
5. 「総合原価計算の月末仕掛品材料費の計算に加工進捗度を使う」→ **誤り**（始点投入の場合は1.0）
