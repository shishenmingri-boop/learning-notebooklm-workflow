# Daily News Digest - 2026-06-23

BBC NewsとAl Jazeeraの主要ニュースを日本語で確認するための1日分パッケージです。

## 概要

- 実行日: 2026-06-23
- 対象媒体: BBC News, Al Jazeera
- 目的: 毎朝のDaily News Digest運用の手動検証

## ファイル

- `sources.md`: 採用した記事URLと短い要約
- `00_overview.md`: 全体サマリー
- `01_bbc.md`: BBC記事の翻訳・要約
- `02_aljazeera.md`: Al Jazeera記事の翻訳・要約
- `03_sources.md`: 情報源一覧と採用メモ
- `notebooklm-prompt.md`: NotebookLMに貼る分析プロンプト
- `notebooklm-upload-plan.md`: NotebookLM投入手順

## 注意

- BBC RSSは500エラー、Al Jazeera RSSはタイムアウトのため、公開ニュースページと記事URLから取得した。
- Al Jazeeraは2026-06-23付き新規記事が確認できなかったため、2026-06-22公開の関連記事を本日の論点として採用している（`03_sources.md` 参照）。
